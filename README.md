# Juno CRM Project

This project was coded with Angular 4 (http://juno.puxo.net) version 1.0.0.

## Installation

Run `npm install` 

## Start on Local

Run `ng servve` and look at http://localhost:4200

## Build

Run `ng buil -prod --aot`

## Deploy

Run `npm run deploy` . (works only on mac);
